Step-by-Step Optimization with Elpee
====================================

**Elpee** (short for **L** inear **P** rogramming) is a Python library for students and academics to viusalize
the steps and iterations required to solve constrained linear optimization problems using linear programming
methods.

Check out the :doc:`usage` section for further information, including
how to :doc:`install` the project.

.. note::

   This project is under active development.
