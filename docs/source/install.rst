Installation
============

Using PyPi
~~~~~~~~~~

To use elpee Python library, first install it using pip:

.. code-block:: console

   (.venv) $ pip install elpee

From Source Code
~~~~~~~~~~~~~~~~

Clone the `GitHub Project Repo <https://github.com/navindu-ds/elpee>`_ using git

.. code-block:: console

   $ git clone https://github.com/navindu-ds/elpee.git
